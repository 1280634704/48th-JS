02-初识JavaScript

srcipt标签：

1.作用：在HTML页面中插入一段JavaScript代码。
2.标签类型：双标签（注意闭合标签）。
3.使用方法：

<style>

    #box{

        width:100px;

        height:100px;

    }

</style>

<div id='box'></div>

<srcipt>

    document.getElementById('box')=.innnerHTML='请点击';//整行加起来是一句JS代码

</srcipt>

引用JS代码的方式：

1.行内样式：在元素内添加事件，如：

<div onclick="this .immerHTML='按钮'"></div>

2.外联样式：通过软件创建.js文件，再经过srcipt标签引入.js文件，引入方式如下：

<srcipt src='xxx.js'></srcipt>

注意：引入.js文件的srcipt标签就 不能继续引入内嵌样式的JS代码。
3.内嵌样式：在srcipt标签之间写入JS代码，如：

<style>

    #box{

        width:100px;

        height:100px;

    }

</style>

<div id='box'></div>

<srcipt>

    document.getElementById('box')=.innnerHTML='请点击';//整行加起来是一句JS代码

</srcipt>

转义符：

1.定义：将特殊符号转化为普通文本字符。
2.用法：在特殊符号前面加/ 每一个想要表现出来的特殊符号的前面都需要加，例如： alert('这里面想要出现个引号 /" ');。
注意：转义符仅限在一串字符中使用。

弹窗与调试

1.alert属性：设置网页弹窗，如：alert('我就是个弹窗');。
2.console.log（xxx）：日志打印，在F12中的console界面中能找到数据，相对于alert属性来说，不会阻止下面的代码的执行
注意：用来调试的语句在demo上线的时候需要备注掉。

弹窗的三种方式：

1.alert：设置网页弹窗，只有一个确定选项。用法如下：alert('我就是个弹窗');。
2.confirm：网页弹窗，有确定和取消选项，根据用户的选择返回真/假值。用法:confirm('第二种弹窗哦')。
3.prompt：网页弹窗，有文本出入框和确定、取消按钮，可以返回用户填写的内容或是none值。用法：prompt('你想写点啥？')；。
4.缺陷：由于是系统弹窗软件因此会有两个缺点：① 样式不统一：不同浏览器的样式是不同的；②格式不美观。因此在工作中需要用到弹窗的时候就会用一个div来自定义弹窗效果。

点击事件和innerHTML属性：

1.获取元素：

  • 通过Id选择器获取元素：document.getElementById(''); 全浏览器兼容。
  • 通过class选择器获取元素：document.getElementByClassName(''); 获取一组元素，不支持IE8以下的浏览器；
  • 通过标签名获取元素：document.getElementByTagNamen(''); 通过标签名获取，获取一组元素，全浏览器兼容；
  • 通过name获取元素：document.getElementByName(''); 通过name获取一组元素，一般很少用，全浏览器兼容；
  • 通过CSS选择器获取：document.getElementBySelector(''); 通过CSS选择器获取，获取第一个元素，支持IE8以上；
  • 通过CSS选择器选择一组元素：docunment.getElementBySelectorAll(''); 获取满足这个选择器的所有元素，支持IE8及以上；
    2.反馈机制：
    ①点击事件：document.getElementById(‘box’) .onclick = function(){插入JS代码；};
    ②innerHTML属性：更改元素中的内容 用法：document.getElementById(‘box’) .innerHTML=’点击‘；

定义变量

  1. ES6新版变量定义：
    • let：定义变量，用法：let x = 10;
    • const：定义常数变量，用法：const x = 10;
      • 注意：常数变量必须有初始值，哪怕是underfined(未定义)。
    • function：定义函数
      注意：ES6对IE8以下的浏览器不兼容；
  2. ES5定义变量：
    • var：定义变量，相当于let;
    • class:定义一类，不能定义其他类型，比如：定义变量；
    • function：定义函数；

注意：在写JS代码的时候不要将ES5和ES6穿插使用。

  1. 变量命名规则：
    • 不能使用关键词命名，例如：let、const (JS内有意义的词)；
    • 不能使用保留词命名，例如：float (JS内以后可能会有意义的词)；
    • 组成命名的词最好是：英文、数字、下划线_ 、美元符号 $组成，且不能以数字作为开头；
    • 命名要见名知意；
    • 命名节点是要在前面加小写的“o”；
    • 驼峰命名法：当有多个单词组成一个命名时除首个单词之外以后每个单词的首字母需要大写，如：let shopCar = 10；
