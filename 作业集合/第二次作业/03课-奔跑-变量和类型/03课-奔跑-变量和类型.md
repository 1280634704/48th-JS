# 定义变量

命名注意点:

1.驼峰命名法

2.不能定义JS里有意义的语法

3.不能定义关键词

4.可以用字母,数字,$,_,命名  数字不能当开头

5.通透话让人一目了然

6.定义变量名时把变量名都统一的放在前面一起



# ES5跟ES6

**ES5声明变量的方法:**

var	声明变量

function	声明函数



**ES6声明变量的方法:**

let	声明变量

const	声明初始变量

function	声明函数

# 数据的类型

1.undefined	定义的变量没有赋值

2.number	数值0.1.2等等

3.string		字符串,可以做拼接用

4.null		空值 

5.object		(除了布尔值外)数据组:数组,对象,函数,json 

6.molean	布尔值(true跟false)
