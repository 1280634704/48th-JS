EASCScript 简称ES ，ES5之前变量声明使用 ：var

var声明的变量存在变量提升问题，ES5前只有 全局变量 和 函数变量 两种

、

1.方法中未定义的变量会变成 全局变量，即window顶层对象，会污染全局环境

2.var声明的全局全量可以在赋值后定义，但不符合逻辑，约定变量先声明后使用


ES6之后声明使用 let：变量，const:常量

ES6之后的变量不会成为Windows顶层对象

ES执行模式分为标准模式和严格模式，ES默认执行标准模式，严格模式主要是解决ES5的漏洞


数据类型

ES5之前数据类型有6种，ES6新增1总数据类型symbbol

1.Number:数值型数据，包括NaN非数值，正负无穷

2.String：字符型数据，用引号包裹，表示字符串

3.undefined:表示未定义，当数据使用为赋值，默认值就是undefined

4.null：空类型表示没有

5.boolean：布尔型表示逻辑0代表false ，1代表true

Object引用型数据类型:表示对象，Object数据类型分类

1.函数function ：function fn（）{}

2.Json 键值对： let Object = { name:"乌拉", age:28, method:function fn(){}, faimly:["父母","兄弟","姐妹","七大姑八大婆"] }

2.日期Date 3.数组 Array4.正则RegExp
